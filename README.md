#Cajero Automático en C++

Simulación de un cajero automático en consola desarrollado en C++.

##Funcionalidades del programa: 

- Inicio de sesión con PIN
- Consulta de saldo
- Depósitos
- Retiros
- Guardado automático del saldo en archivo

## Tecnologías usadas
- C++
- Manejo de archivos (fstream)

##Forma de ejecución: 

```bash
g++ main.cpp -o cajero
./cajero
